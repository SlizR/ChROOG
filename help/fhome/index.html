How It Works

  1. UI Rendering: Sidebar (chat list, new chat button, settings), Main area (messages), Settings modal.
  2. Chat Handling: Chats are stored as objects in local memory and cookies.
  3. Saving to Cookies: setCookie('fhomeai_data', JSON.stringify({chats, settings}), 365); ensures fully local and privacy-friendly operation.
  4. AI Request Flow: User messages are wrapped in instruction blocks, sent to Gemini API, and responses appended to chat with typing indicator.

API Requirements

You need a Google AI Studio API key: https://aistudio.google.com. Update in code with: const API_KEY = "YOUR_API_KEY_HERE";
